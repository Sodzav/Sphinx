
==============
Каталог
==============

На страницуе каталога отображается:
* Фильтр
* Товар

------------------
Фильтр
------------------

Фильтр помогает в поиске определенного товара, он подразделяется на:
* Вид товара
* Тип товара 
* модель товара
* Цена товара
* Магазин

-----------
Вид товара
~~~~~~~~~~~

1. Нажимаете на кнопку в виде стрлки вниз
2. Выберите интересующий вас вид товара
3. Нажмите кнопку "отсортировать"


Тип товара
~~~~~~~~~~

Если есть подразделение продукта по типу
1. Нажмите на кнопку в виде стрелки вправо
2. Выберите интересующий вас тип товара
3. Нажмите кнопку "отсортировать"


Модель товара
~~~~~~~~~~~~~

1. Поставте галочки в поле интересующих вас моделей
2. Нажмите кнопу "отсортировать"


Цена товара
~~~~~~~~~~~
1. Нажмите на вкладку фильтра цена товара
2. В появившемя окне укажите цену "от" и "до"
3. Либо перетащите палзунок на нужное значение
4. Нажмите кнопку "отсортировать"


Магазин
~~~~~~~~~~
1. Нажмите на вкладку фильтра "Магазин"
2. выберите интересующие вас Магазины 
3. Нажмите кнопку "отсортировать"


Товар
------
1. Выберите интересующий вас товар из предложенного списка 
2. Нажмите кнопку "перейти на сайт" для дальнейшего заказа.

`Вернуться к списку разделов <index.html>`_
